**修复前**: 只显示"导出功能 (演示)"通知

**修复后**: 真实导出CSV文件（Excel可打开）

**功能特性**:
- ✅ 自动获取当前表格数据
- ✅ 导出为CSV格式（Excel兼容）
- ✅ 支持中文（添加BOM）
- ✅ 自动生成文件名（包含时间戳）
- ✅ 处理特殊字符（逗号、引号、换行）
- ✅ 显示导出成功通知（包含数据条数）

**代码实现**:
```javascript
else if (text === '导出') {
  // 真实导出Excel功能
  const panel = btn.closest('.panel');
  if (panel) {
    const table = panel.querySelector('table');
    if (table) {
      // 获取表格数据
      const rows = table.querySelectorAll('tr');
      let csvContent = '';
      
      rows.forEach((row, index) => {
        const cells = row.querySelectorAll(index === 0 ? 'th' : 'td');
        const rowData = Array.from(cells).map(cell => {
          // 清理单元格内容
          let text = cell.textContent.trim();
          // 如果包含逗号或换行，用引号包裹
          if (text.includes(',') || text.includes('\n') || text.includes('"')) {
            text = '"' + text.replace(/"/g, '""') + '"';
          }
          return text;
        }).join(',');
        csvContent += rowData + '\n';
      });
      
      // 添加BOM以支持中文
      const BOM = '\uFEFF';
      const blob = new Blob([BOM + csvContent], { type: 'text/csv;charset=utf-8;' });
      
      // 生成文件名
      const panelTitle = panel.querySelector('.panel-title')?.textContent.trim() || '数据';
      const timestamp = new Date().toISOString().slice(0, 19).replace(/:/g, '-');
      const filename = `${panelTitle}_${timestamp}.csv`;
      
      // 创建下载链接
      const link = document.createElement('a');
      link.href = URL.createObjectURL(blob);
      link.download = filename;
      link.style.display = 'none';
      document.body.appendChild(link);
      link.click();
      document.body.removeChild(link);
      
      // 显示成功通知
      showNotification(`✅ 已导出 ${rows.length - 1} 条数据`, 'success');
    }
  }
}
```
