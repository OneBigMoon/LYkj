## 五、节点配置测试 ✓

### 测试步骤:
对每种节点类型进行配置测试:

#### 5.1 数据源节点
1. 点击"配置"按钮
2. 选择数据源类型
3. 填写连接地址
4. 填写数据表名
5. 点击"保存配置"

#### 5.2 数据处理节点
1. 点击"配置"按钮
2. 选择处理类型
3. 填写处理脚本
4. 设置批量大小和并行度
5. 点击"保存配置"

#### 5.3 模型训练节点
1. 点击"配置"按钮
2. 选择模型类型
3. 设置批量大小、学习率
4. 设置最大Epoch、GPU数量
5. 选择优化器
6. 点击"保存配置"

#### 5.4 模型评估节点
1. 点击"配置"按钮
2. 勾选评估指标
3. 填写测试数据集
4. 设置通过阈值
5. 点击"保存配置"

#### 5.5 模型部署节点
1. 点击"配置"按钮
2. 选择部署环境
3. 选择部署策略
4. 设置灰度比例
5. 设置实例数量和CPU核心
6. 点击"保存配置"

#### 5.6 条件分支节点
1. 点击"配置"按钮
2. 选择条件类型
3. 填写条件表达式
4. 查看示例提示
5. 点击"保存配置"

#### 5.7 循环节点
1. 点击"配置"按钮
2. 选择循环类型
3. 填写循环次数/条件
4. 设置最大迭代次数
5. 点击"保存配置"

#### 5.8 定时触发节点
1. 点击"配置"按钮
2. 选择触发类型
3. 填写时间配置
4. 选择时区
5. 查看Cron示例
6. 点击"保存配置"

### 预期结果:
- ✓ 配置弹窗正确打开
- ✓ 所有表单字段正常显示
- ✓ 下拉框可以选择
- ✓ 输入框可以输入
- ✓ 保存后显示"配置已保存"
- ✓ 弹窗自动关闭
