# 移动智能语音AI训练中台 - 功能说明文档

**项目名称**: 移动智能语音AI训练中台  
**文档版本**: V1.0  
**创建日期**: 2026-03-03  

---

## 模块1: 智能指挥中心

### 功能描述
企业级AI运营总览大屏,提供全局视角的实时监控和数据分析,帮助管理者快速掌握平台运营状态、模型性能、系统告警等关键信息。

### 功能需求
1. 核心指标监控:训练任务数、模型准确率、数据资产、在线用户数
2. 生产模型实时监控:展示各行业具体模型(金融-ASR、电商-NLU等)的QPS、响应时间、准确率、调用量
3. AI通话实时监控:今日通话总数、AI自动处理vs人工接管比例、实时通话列表、客户满意度
4. 人工核对待办:待核对标注数据、待审批模型发布、待处理异常通话,按优先级分类
5. 系统告警中心:服务器离线、硬盘使用率、待核对积压、数据库连接池等运维告警

**截图位置**: [打开系统默认首页]

## 模块2: 账号权限管理

### 功能描述
企业级RBAC权限体系,提供完整的用户生命周期管理和细粒度权限控制,支持多角色、多层级的权限体系。

### 功能需求
1. 用户管理:用户列表展示、搜索筛选、新建编辑、批量操作、导入导出
2. 用户信息:用户名、姓名/工号、角色、部门、状态、最后登录时间和IP
3. 角色权限配置:超级管理员、模型工程师、数据分析师、标注员等角色,支持128项细粒度权限
4. 权限分类:AI模型管理、特征工程、数据治理、用户管理、审计中心等

**截图位置**: [点击侧边栏"账号权限管理"]

## 模块3: AI模型工厂

### 功能描述
企业级MLOps核心引擎,提供完整的模型训练生命周期管理,从任务创建、实时监控到模型注册。

### 功能需求
1. 统计概览:训练任务数、模型注册数、实验追踪数、成功率统计
2. 训练任务流水线:运行中任务(显示进度、Loss、Acc、资源使用)、排队中任务(队列位置、预计等待)、已完成任务(最终指标、性能提升)
3. 任务控制:查看详情、TensorBoard、暂停、停止、调整资源、调整优先级
4. 训练模式支持:微调、全量训练、增量训练、迁移学习

**截图位置**: [点击侧边栏"AI模型工厂"]

## 模块4: 实验追踪中心

### 功能描述
实验管理和对比分析平台,自动记录实验参数和结果,支持多实验对比,帮助快速找到最优模型配置。

### 功能需求
1. 实验列表:实验ID、名称、模型类型、准确率、Loss、状态、创建时间
2. 实验对比:准确率对比柱状图、趋势分析、最优实验识别
3. 实验记录:自动记录实验参数、训练指标、模型产物
4. 操作功能:创建新实验、对比实验、查看详情、导出结果

**截图位置**: [点击侧边栏"实验追踪中心"]

## 模块5: 知识图谱

### 功能描述
业务知识可视化管理平台,将业务知识结构化为图谱形式,支持实体和关系的可视化管理。

### 功能需求
1. 知识图谱可视化:SVG交互式图谱、多层级结构(用户-业务-意图-实体)、关系展示、支持缩放拖拽
2. 实体统计:用户、业务、意图、实体等类型统计,显示数量和占比
3. 关系统计:拥有、关联、包含、依赖等关系统计,显示数量和占比
4. 三元组管理:总三元组数统计、支持查询和编辑

**截图位置**: [点击侧边栏"知识图谱"]

## 模块6: 链路追踪

### 功能描述
分布式调用链监控平台,提供完整的调用链追踪能力,帮助快速定位性能瓶颈和故障点。

### 功能需求
1. 实时监控指标:今日调用量、平均耗时、成功率、错误数
2. 调用链列表:Trace ID、服务链路(API→ASR→NLU→Dialog→RAG→Response)、总耗时、Span数、状态
3. 服务性能分析:各服务平均耗时柱状图、最慢服务识别、瓶颈定位
4. 性能优化建议:基于分析结果提供优化建议

**截图位置**: [点击侧边栏"链路追踪"]

## 模块7: 数据治理

### 功能描述
企业级数据全生命周期管理平台,提供数据资产管理、数据质量监控、数据血缘追溯能力。

### 功能需求
1. 数据资产统计:数据源、数据集、数据表、标签的数量统计
2. 数据源管理:数据源列表(MySQL、OSS等)、状态监控、表级详情(记录数、大小、更新时间)
3. 数据血缘追溯:上游依赖分析、下游影响分析、完整血缘链路、影响范围评估
4. 数据质量监控:质量得分、异常数据统计、质检任务、4个质量维度评分(完整性、准确性、一致性、及时性)、近30天质量趋势

**截图位置**: [点击侧边栏"数据治理"]

## 模块8: 智能标注

### 功能描述
AI辅助的智能标注平台,采用"AI预标注+人工修正"模式,大幅提升标注效率,同时保证标注质量。

### 功能需求
1. 标注统计:标注任务数、AI标注率、人工修正率、标注准确率
2. 标注任务列表:任务名称、类型(意图分类/实体识别)、总数、AI标注数、人工修正数、已完成数、状态
3. AI vs 人工对比:速度、成本、准确率的对比分析
4. 智能建议:基于数据提供标注策略建议

**截图位置**: [点击侧边栏"智能标注"]

## 模块9: 模型监控

### 功能描述
生产模型性能监控平台,实时监控生产环境中所有模型的性能指标,及时发现异常。

### 功能需求
1. 监控统计:生产模型数、平均响应时间、平均准确率、漂移告警数
2. 模型性能监控表:模型名称和版本、QPS、响应时间、准确率、健康度状态
3. 模型性能趋势:准确率趋势图(近7天)、稳定性分析
4. 性能亮点:展示表现优异的模型及其关键指标

**截图位置**: [点击侧边栏"模型监控"]

## 模块10: 人工干预

### 功能描述
AI记录+人工处理工单系统,AI自动识别问题并生成工单,人工处理后反馈优化模型,形成闭环。

### 功能需求
1. 工单统计:待处理工单数、AI自动记录数、已处理数、平均处理时长
2. 工单列表:工单ID、问题类型(用户投诉/标注修正/模型优化/数据质量)、问题描述、优先级、AI置信度、创建时间、状态
3. 工单统计分析:按问题类型统计、处理效率统计(平均处理时长、24小时内处理率、用户满意度、重复工单率)
4. 工单流程:AI识别→生成工单→分配处理→人工处理→结果反馈→模型优化

**截图位置**: [点击侧边栏"人工干预"]

## 模块11: 审计中心

### 功能描述
操作审计和日志管理平台,记录系统所有关键操作,提供完整的审计追踪能力,确保操作可追溯。

### 功能需求
1. 审计统计:今日操作数、活跃用户数、异常操作数、合规率
2. 操作审计日志表:时间、用户、操作类型、操作对象、风险等级(高/中/低)、状态
3. 详细操作日志:完整的操作记录,包含IP地址、审批单号、操作参数、执行结果等详细信息
4. 日志筛选:支持按时间、用户、操作类型、风险等级筛选

**截图位置**: [点击侧边栏"审计中心"]

## 模块12: 流程编排

### 功能描述
可视化流程编排平台,支持拖拽式设计复杂业务流程,包含数据处理、模型训练、模型部署等节点。

### 功能需求
1. 流程统计:流程总数、成功率、平均耗时、今日执行数
2. 流程编排画布:可视化拖拽设计、支持多种节点类型(数据源、数据处理、模型训练、模型评估、模型部署、条件分支、循环、定时触发)
3. 流程列表:流程名称、类型、节点数、状态、最后执行时间、成功率
4. 流程控制:保存、运行、新建流程

**截图位置**: [点击侧边栏"流程编排"]

## 模块13: 对话测试

### 功能描述
对话测试台,支持实时测试AI模型的对话能力,提供实时分析和调用链路展示。

### 功能需求
1. 模型选择:支持切换不同行业模型(金融-ASR、电商-NLU、政务-意图、医疗-随访)
2. 对话界面:AI和用户的对话展示、显示置信度和意图识别结果
3. 实时分析:意图识别、实体抽取、情感分析、响应时间
4. 调用链路:显示完整的服务调用链(ASR识别→NLU理解→对话管理→业务查询)
5. 操作功能:清空对话、导出记录

**截图位置**: [点击侧边栏"对话测试"]

## 模块14: 系统配置

### 功能描述
系统配置管理平台,提供系统基础配置、功能开关、参数设置等管理能力。

### 功能需求
1. 配置分类:基础配置、模型配置、告警配置、安全配置、集成配置
