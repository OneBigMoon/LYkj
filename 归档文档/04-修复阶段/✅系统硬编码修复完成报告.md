## 🎨 技术实现

### 统一的修复模式

所有页面都遵循相同的修复模式：

```javascript
// 1. 为指标元素添加ID
<div class="metric-value" id="page_metricName">硬编码值</div>

// 2. 添加或修改onMount函数
onMount() {
  // 从VirtualDB读取数据
  const data = VirtualDB.getTable('tableName');
  
  // 计算指标
  const metricValue = data.filter(...).length;
  
  // 更新DOM
  const el = document.getElementById('page_metricName');
  if (el) el.textContent = metricValue;
}
```

### ID命名规范

采用统一的ID命名规范：`{页面前缀}_{指标名称}`

- dashboard: 无前缀（taskCount, onlineUsers等）
- model-factory: mf_前缀
- trace: trace_前缀
- data-governance: dg_前缀
- labeling: label_前缀
- model-monitor: mm_前缀
- intervention: iv_前缀
- flow: flow_前缀
