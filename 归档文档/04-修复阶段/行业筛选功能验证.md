### 3. 实现各页面的筛选函数 ✅

#### 3.1 实验跟踪页面筛选
```javascript
function filterExperimentByIndustry(industry) {
  const table = document.getElementById('experimentTable');
  if (!table) return;
  
  const rows = table.querySelectorAll('tbody tr');
  let visibleCount = 0;
  
  rows.forEach(row => {
    const experimentName = row.cells[1]?.textContent || '';
    
    if (industry === 'all') {
      row.style.display = '';
      visibleCount++;
    } else {
      // 根据实验名称判断行业
      const industryMap = {
        'finance': '金融',
        'ecommerce': '电商',
        'gov': '政务',
        'medical': '医疗'
      };
      
      if (experimentName.includes(industryMap[industry])) {
        row.style.display = '';
        visibleCount++;
      } else {
        row.style.display = 'none';
      }
    }
  });
  
  // 重新应用分页
  if (paginationState['experimentTable']) {
    paginationState['experimentTable'].currentPage = 1;
    applyPagination('experimentTable');
  }
}
```

function filterDashboardByIndustry(industry) {
  const models = document.querySelectorAll('[style*="margin-bottom: 8px;"] > div > span[style*="font-size: 14px;"]');
  
  models.forEach(model => {
    const parent = model.closest('[style*="margin-bottom: 16px;"]') || model.closest('[style*="margin-bottom: 12px;"]');

## 💡 使用说明

### 如何使用行业筛选

1. **查看全部数据**
   - 点击顶部的"全部"按钮
   - 所有行业的数据都会显示

2. **筛选特定行业**
   - 点击顶部的行业按钮（金融/电商/政务/医疗）
   - 只显示该行业的数据
   - 其他行业的数据会被隐藏

3. **切换页面**
   - 切换到其他页面时，筛选状态会保持

4. **查看通知**
   - 每次切换行业时，会显示成功通知
