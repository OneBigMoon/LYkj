# 流程编排功能 - 快速使用指南

## 🚀 快速开始

### 1. 打开演示
在浏览器中打开 `ai-platform-complete.html`，点击左侧菜单的"流程编排"。

### 2. 创建第一个流程
1. 从左侧节点库拖拽"数据源"到画布
2. 拖拽"模型训练"到画布
3. 从"数据源"的右侧圆点拖拽到"模型训练"的左侧圆点
4. 点击"运行"按钮查看执行动画

---

## 📋 节点类型

### 数据节点 (蓝色)
- 📊 **数据源**: 配置数据库或文件数据源
- ⚙️ **数据处理**: 配置数据清洗和转换

### 模型节点 (绿色)
- 🎯 **模型训练**: 配置训练参数和资源
- 📈 **模型评估**: 配置评估指标和阈值
- 🚀 **模型部署**: 配置部署环境和策略

### 控制节点 (橙色)
- 🔀 **条件分支**: 配置条件表达式
- 🔁 **循环**: 配置循环次数或条件
- ⏰ **定时触发**: 配置Cron表达式

---

## 🎯 核心操作

### 添加节点
拖拽左侧节点库中的节点到画布

### 移动节点
点击并拖拽节点到新位置

### 连接节点
从输出点(右侧圆点)拖拽到输入点(左侧圆点)

### 配置节点
点击节点上的"配置"按钮

### 删除节点
点击节点上的"删除"按钮

### 删除连接
点击连接线，确认删除

---

## 🔧 画布操作

### 清空画布
点击顶部"清空"按钮，确认清空所有节点和连接

### 保存流程
点击顶部"保存"按钮，流程数据会输出到控制台

### 运行流程
点击顶部"运行"按钮，观察执行动画

---

## 🔍 流程列表

### 搜索流程
在搜索框输入关键词，实时过滤流程列表

### 筛选流程
点击"全部"/"运行中"/"已完成"按钮筛选

### 查看流程
点击"查看"按钮查看流程详情和执行历史

### 编辑流程
点击"编辑"按钮修改流程配置

### 复制流程
点击"复制"按钮创建流程副本

---

## ⚠️ 常见问题

### Q: 为什么不能运行流程?
A: 检查是否有孤立节点或循环依赖，系统会自动提示错误

### Q: 如何删除连接?
A: 直接点击连接线，在确认对话框中选择"确定"

### Q: 节点配置保存在哪里?
A: 配置保存在内存中，点击"保存"按钮可以输出到控制台

### Q: 如何创建分支流程?
A: 使用"条件分支"节点，从该节点可以连接到多个后续节点

### Q: 如何创建循环流程?
A: 使用"循环"节点，但注意系统会检测循环依赖

## 📊 示例流程

### 示例1: 简单训练流程
```
数据源 → 数据处理 → 模型训练 → 模型评估 → 模型部署
```

### 示例2: 条件分支流程
```
数据源 → 模型训练 → 模型评估 → 条件分支
                                    ├→ 模型部署 (准确率>95%)
                                    └→ 模型优化 (准确率≤95%)
```

### 示例3: 循环训练流程
```
数据源 → 模型训练 → 模型评估 → 条件分支
            ↑                        ↓
            └────── 循环 ←───────────┘
                  (准确率<95%)
```

### 示例4: 定时任务流程
```
定时触发 → 数据源 → 数据处理 → 模型训练
(每天2点)
```
