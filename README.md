## 📊 核心功能模块

### 1. 智能指挥中心 (Dashboard)
- AI通话实时监控（通话时长每秒跳动）
- 生产模型性能监控（QPS/响应时间实时波动）
- 系统告警实时生成
- 人工核对待办动态更新

### 2. 账号权限管理 (User Management)
- 用户在线状态每20秒自动切换
- 角色权限配置
- 用户增删改查

### 3. AI模型工厂 (Model Factory)
- 训练任务进度实时增长
- Loss/Acc实时更新
- 实验追踪中心
- 模型注册管理

### 4. 链路追踪 (Trace)
- 每10秒生成1-2条新链路
- 6种服务链路模板
- 95%成功率，智能时间显示

### 5. 数据治理 (Data Governance)
- 数据质量评分每15秒动态更新
- 4个质量维度（完整性、准确性、一致性、及时性）
- 数据血缘图谱

### 6. 智能标注 (Labeling)
- 标注任务进度每5秒更新
- 自动状态转换（进行中→待审核）
- 50个标注任务

### 7. 模型监控 (Model Monitor)
- 模型性能指标实时波动
- 响应时间颜色动态变化
- 告警规则配置

### 8. 人工干预 (Intervention)
- 工单状态每20秒自动流转
- 新工单每30秒30%概率生成
- 8种问题类型，4个优先级

### 9. 审计中心 (Audit)
- 操作自动记录到审计日志
- 审计日志每5秒刷新
- 支持高级搜索和数据联动
